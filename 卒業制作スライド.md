# フロント
## Vue.js + vue-router + Vuex
- 外観はVuetify(Material Component Framework)
- API叩いてJSON受け取って表示していく
- **vue-router** で画面遷移させSPAを実現
- **Vuex** でState(状態)管理、APIを叩くactionを実装
